"# Data" 
📊 Sales Analysis Dashboard (Streamlit)
📌 Overview

This project is an interactive Sales Analysis Dashboard built using Python and Streamlit. It helps users visualize and analyze sales data in a simple and user-friendly way. The dashboard provides insights into sales performance by country, product, and other key metrics.

🚀 Features
📈 Interactive data visualization
🌍 Country-wise sales analysis
📊 Pie charts, bar charts, and more
🔍 Dynamic filtering of data
⚡ Fast and lightweight UI using Streamlit
🛠️ Technologies Used
Python 🐍
Streamlit
Pandas
Plotly
📂 Project Structure
├── app.py              # Main Streamlit application
├── data.csv            # Dataset used for analysis
├── requirements.txt    # Required Python libraries
└── README.md           # Project documentation
▶️ How to Run the Project
Clone the repository:
git clone https://github.com/your-username/sales-dashboard.git
Navigate to the project folder:
cd sales-dashboard
Install dependencies:
pip install -r requirements.txt
Run the Streamlit app:
streamlit run app.py

Country-wise sales distribution
Product performance comparison
Monthly or yearly sales trends
